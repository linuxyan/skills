# Phase 2 特殊处理详细规则

本文档包含 Phase 2 中三个特殊子阶段的详细规则：100%分红率处理、周期行业适配、银行股专属修正。

---

## Phase 2.1：100% 分红率特殊处理

核心问题：标准公式 `g_base = ROE × (1 - Payout)` 中，`(1 - Payout)` 趋于 0，导致 g_base ≈ 0，严重低估了顶级品牌公司通过定价权驱动的真实增长能力。

### Step C：确定公司竞争等级（定价权评估）

根据以下三个维度综合判断（满足任意两个即可归入该档）：

| 维度 | TOP | STRONG | NORMAL |
|------|-----|--------|--------|
| 市占率/竞争格局 | 绝对垄断（>70%）或品类=品牌 | 龙头地位稳固（30-70%），有明确护城河 | 竞争激烈，无明显护城河 |
| 定价权 | 有持续提价能力且历史兑现（如茅台/片仔癀） | 有一定议价能力，提价偶尔兑现 | 无定价权，价格战频发 |
| ROE 水平 | ROE ≥ 20%，且稳定性强 | ROE 15-20%，或 ≥ 20% 但波动大 | ROE < 15% 或 ROE 波动极大 |
| 典型案例 | 茅台、片仔癀、东阿阿胶 | 腾讯、格力电器 | 大多数普通消费品 |

### Step D：查表确定 g_floor

| 竞争等级 | g_floor（保底增速） | 必要条件（ROE） | 充分条件（定价权/护城河） |
|---------|-------------------|---------------|------------------------------|
| **TOP-4%** | **4%**（极强定价权） | ROE ≥ 25%，且稳定性强 | 有持续提价能力且历史兑现；或品类=绝对品牌垄断 |
| **TOP-3%** | **3%**（稳健） | ROE ≥ 20%，且稳定性强 | 品类=品牌，市占率>70%；或具备较强定价权 |
| **STRONG-2%** | **2%**（较强惯性） | ROE 20-25%，或 ≥25% 但波动较大 | 龙头地位稳固（30-70%），提价偶尔兑现；有明确护城河 |
| **STRONG-1.5%** | **1.5%**（稳健） | ROE 15-20% | 有一定议价能力，行业惯性较强 |
| **NORMAL** | **0%** | 不满足以上条件 | 竞争激烈，无明显护城河，无定价权 |

> **g_floor 选取规则**：按从上到下顺序匹配，优先尝试更高档。ROE 是基础门槛，但**定价权/护城河是升档的充分条件**。

### Step E：红旗检测——预收/合同负债连续两季同比下滑

> **适用条件**：当公司被识别为 TOP 或 STRONG 档时执行，NORMAL 档跳过。

预收款项/合同负债是品牌消费品（尤其是白酒）的需求先行指标。若该指标连续两季同比下滑，说明渠道/消费者打款意愿减弱。

```
Step E1：获取近4季预收/合同负债数据
        # 预收款项（老准则）和合同负债（新准则）是同一科目，合并查询
        查询关键词："{公司名}{代码} {年份}预收款项合同负债科目余额同比变动季报"

Step E2：判断是否连续两季同比下滑
        if 近2季同比均 < 0：
            红旗触发 = True
            "预收/合同负债连续两季同比下滑（Q1:{X}%，Q2:{X}%），g_floor补偿取消"
        else：
            红旗触发 = False
            "预收/合同负债无连续下滑信号，g_floor补偿维持"

Step E3：确定 g_floor_final
        if 红旗触发：g_floor_final = 0
        else：g_floor_final = g_floor（沿用Step D结果）
```

**报告标注**（必须在估值计算过程中展示）：
```
| 预收/合同负债趋势 | Q1同比 {+X%/-X%}，Q2同比 {+X%/-X%} | 红旗信号：{触发/未触发} |
| g_floor 最终取值 | {X}%（原始值 {X}%，红旗影响：{取消/维持}） | |
```

### Step F：现金流溢价系数（⚠ 仅对 TOP 档公司执行）

> **适用条件**：仅当竞争等级为 TOP 档时执行。STRONG 档及以下不适用。

若 OCF/净利润 ≥ 1.1，说明公司利润几乎全部转化为现金，安全买入价应给予上浮。

```
Step F1：计算近3年平均 OCF/净利润
        OCF_净利润比 = 近3年平均经营现金流净额 ÷ 近3年平均归母净利润

Step F2：判断溢价档位
        if OCF_净利润比 ≥ 1.1：
            现金流溢价系数 = 10%
        elif OCF_净利润比 ≥ 1.0：
            现金流溢价系数 = 5%
        else：
            现金流溢价系数 = 0%

Step F3：计算含溢价安全买入价
        安全买入价（含现金流溢价） = 安全买入价（DDM基础） × (1 + 现金流溢价系数)
```

**报告标注**：
```
| OCF/净利润均值（近3年） | X.XX | ≥1.1=溢价10%，≥1.0=溢价5%，<1.0=无溢价 |
| 现金流溢价系数 | +{X}% | {说明} |
| 安全买入价（含现金流溢价） | ¥XX.XX | DDM基础价 ¥XX × (1+{X}%) |
```

### Step G：最终 g 取值

```
g_final = max(g_base, g_floor_final)
# 实际效果：
# - TOP 公司：g_final = max(≈0, 3-4%) = 3-4%（红旗触发则清零）
# - STRONG 公司：g_final = max(≈0, 1.5-2%) = 1.5-2%
# - NORMAL 公司：g_final = 0%
# 注意：最终仍需与 ROE 档位 Cap 比对，取较小值
```

### Phase 2.1 完整执行流程

```
Step A：计算标准 g_base
Step B：Payout ≥ 95%？ → 是：进入 Step C；否：标准处理 → Phase 2.5
Step C：确定竞争等级（TOP/STRONG/NORMAL）
Step D：查表确定 g_floor
Step E：红旗检测（预收/合同负债连续两季下滑）→ 取消/维持 g_floor → g_floor_final
Step F：现金流溢价系数（仅TOP档）→ 计算溢价幅度
Step G：g_final = max(g_base, g_floor_final) → 与 ROE Cap 取 min
        → 安全买入价（含现金流溢价）= DDM结果 × (1 + 现金流溢价系数)
```

### 报告中的标注要求

当 Payout ≥ 95% 时，在估值计算过程（Step 1）中必须说明：
1. 公司竞争等级判定（TOP / STRONG / NORMAL）及判定依据
2. 选用的 g_floor 值及理由
3. 红旗检测结果（预收/合同负债趋势 + 是否取消 g_floor）
4. 现金流溢价系数（仅TOP档）
5. 最终 g_final 取值及是否触发了 ROE 档位截断

### ⚠ 常见错误警示

| 错误做法 | 后果 | 正确做法 |
|---------|------|---------|
| 对茅台、东阿阿胶等顶级公司直接设 g=0 | 安全买入价严重低估 | 对 TOP 公司使用 g_floor=3-4% |
| 对没有定价权的普通公司也设 g=3% | 安全买入价过于乐观 | 对 NORMAL 公司维持 g=0% |
| 不区分 TOP/STRONG/NORMAL | 分类模糊，结论主观 | 按三维度综合打分 |

## Phase 2.4：周期行业适配增强

> **触发条件**：此步骤**自动对所有股票执行**，通过ROE标准差判断是否触发特殊处理。

### 第一步：计算近 5 年 ROE 标准差

> ⚠ **ROE 类型规定**：统一使用**加权平均净资产收益率**（`roe`），**不得使用摊薄 ROE**。两者差异在 ROE > 20% 时可能超过 2pp。

```
ROE_std = std(加权ROE_年1, 加权ROE_年2, ..., 加权ROE_年5)
```

### 第二步：ROE 异常检测（强制触发）

| 触发条件 | 说明 |
|---------|------|
| **ROE_std ≥ 6 个百分点** | 强周期信号，触发均值回归强制替代 |
| **ROE_std < 6 个百分点** | 无需特殊处理 |

**触发后的 ROE 取值规则（⚠ 仅适用于 ROE_std ≥ 6pp）**：

核心原则：ROE_std ≥ 6pp 说明公司具有强周期性——5年均值可能因行业繁荣顶而虚高，必须取"均值、最近一年、行业中位数"三者最小值作为保守锚。

```
if ROE_std ≥ 6:
    candidates = [均值ROE, 最新年度ROE, 行业中位数ROE]
    ROE_used = min(candidates)
    # 若最小 ROE 为负，用 0 替代
    if ROE_used < 0:
        ROE_used = 0
    # 数据来源：5年均值ROE和最新年度ROE使用加权ROE
    # 行业中位数ROE：查询关键词 "{行业名}行业中位数ROE {年份}"
    # 若无法获取行业中位数，仅用"5年均值"和"最近一年"两者取小值
else:
    ROE_used = 均值ROE（标准处理）
```

**⚠ ROE_std ≥ 6pp 的公司在 Phase 2.5 的④ ROE稳定性维度自动评1分**

### 第三步：周期位置折扣系数

即使 ROE_std < 6pp，若**周期位置处于历史高位**，仍需对 g_base 执行折扣。

**量化指标获取：**

| 指标 | 查询方式 | 数据来源 |
|------|---------|---------|
| 行业产能利用率 | `"{行业}产能利用率 {年份} 过高 历史高位"` | 国家统计局、工信部、行业协会 |
| 行业库存周期 | `"{行业}库存周期 主动补库 被动去库 阶段"` | Mysteel、中国煤炭市场网 |
| 主要商品价格分位数 | `"{商品} 价格 历史分位数 {年份}"` | Wind、Choice、行业协会 |
| 宏观 PMI 库存分项 | `"中国PMI 库存 {年份} {季度}"` | 国家统计局 |

**阶梯折扣系数：**

| 周期位置信号 | 判断依据 | 折扣系数 |
|------------|---------|---------|
| **极度过热（→ 0.5）** | 产能利用率 > 历史90分位；价格处于前10%分位；行业利润率创历史新高 | `g_adjusted = g_base × 0.5` |
| **偏高估（→ 0.7）** | 产能利用率75-90分位；价格60-90%分位；库存周期主动补库顶 | `g_adjusted = g_base × 0.7` |
| **正常/偏低（→ 不打折）** | 产能利用率低于75分位；价格中低位；去库阶段 | 不打折，使用原 g_base |

**⚠ 注意**：周期位置折扣**叠加在 ROE_std 触发规则之后**：
- ROE_std ≥ 6pp：先用保守锚计算 g_base，再判断是否需要折扣
- ROE_std < 6pp：用均值ROE计算 g_base，再判断是否需要折扣

### 第四步：红旗指标——周期性重资产行业高杠杆风险（⚠ ROE_std ≥ 6pp 公司）

> 核心逻辑：在化工、有色、航运等周期性重资产行业，高杠杆是毁灭性风险。

```
Step 4A：判断是否为周期性重资产行业
        if ROE_std ≥ 6pp and 行业属于以下类别：
            化工/有色/航运/钢铁/煤炭/建材/造纸/玻璃
            → 周期性重资产行业标记 = True

Step 4B：获取资产负债率（使用 Phase 1 已获取数据）

Step 4C：判断红旗
        if 资产负债率 > 70%：
            红旗触发 = True
            → Phase 3 "负债率可控"标注"⚠ 严审"，结果降一档
        else：
            红旗触发 = False
```

**红旗触发对 Phase 3 的影响**：

| 资产负债率（周期性重资产） | Phase 3 标注 | 结果 |
|-------------------------|-------------|------|
| ≤ 60% | 正常 | ✅（负债率可控）|
| 60-70% | 观察 | ⚠️（负债率偏高，周期下行有压力）|
| **> 70%** | **⚠ 严审（红旗触发）** | **⚠️（高杠杆+高波动，一票否决风险大）** |

### Phase 2.4 执行流程总结

```
Step 1：计算 ROE_std
Step 2：ROE_std ≥ 6pp？
    → 是：ROE_used = min(均值ROE, 最新年度ROE, 行业中位数ROE) = max(最小ROE, 0)
    → 否：ROE_used = 均值ROE
Step 3：判断行业周期位置
    → 极度过热：g_adjusted = g_base × 0.5
    → 偏高估：g_adjusted = g_base × 0.7
    → 正常：不打折
Step 4：红旗检测（仅 ROE_std ≥ 6pp 的周期性公司）
    → 资产负债率 > 70% → 红旗触发 → Phase 3 严审
Step 5：g_adjusted 与 ROE 档位 Cap 取 min → g_final
Step 6：确定动态目标收益率
Step 7：代入安全买入价公式
Step 8：继续 Phase 2.5 → Phase 2.6 → Phase 2.7 → Phase 3
```

### 动态目标收益率（target_yield_adjusted）

| ROE_std 分档 | 目标收益率 | 逻辑 |
|-------------|----------|------|
| < 3pp（ROE稳定） | **8%（不变）** | 模型输入可靠，无需额外补偿 |
| 3-6pp（ROE波动） | **9-10%** | 轻度风险补偿 |
| ≥ 6pp（强周期） | **10-12%** | 重度风险补偿；与最低ROE锚共同作用 |

### Phase 2.4 执行后的报告标注要求

触发特殊处理时，在"二、估值计算过程"Step 1 中说明：
1. 近5年 ROE 标准差及是否触发均值回归强制替代
2. 若使用了最低ROE（而非均值），说明理由
3. 若使用了行业中位数，说明数据来源
4. 若触发了周期位置折扣，说明具体指标值和折扣系数

## Phase 2.7：银行股专属修正引擎（PB-ROE 联动模型）

> **触发条件**：Phase 2.5 价值路径识别出"PB修复+股息"路径时执行。

### 额外数据需求（银行股专用）

| # | 数据项 | 查询关键词示例 |
|---|--------|--------------|
| **B①** | 资本充足率（核心一级/一级/综合） | `"{银行名}{代码} {年份}核心一级资本充足率 一级资本充足率 资本充足率"` |
| **B②** | 拨备覆盖率 + 不良贷款率 | `"{银行名}{代码} {年份}拨备覆盖率 不良贷款率 贷款拨备率"` |
| **B③** | 近5年 PB 历史数据 | `"{银行名}{代码} 2020-2025历年PB市净率每月历史数据"` |

### 第一步：坏账拨备还原（识别"藏利润"行为）

```
Step A：获取银行拨备覆盖率 + 行业均值
        行业均值参考：五大行均值约 200-250%；全国股份制均值约 180-220%

Step B：判断是否"超额拨备"（藏利润）
        if 拨备覆盖率 > 行业均值 × 1.3（约为 250-300%+）：
            还原比例 = min(50%, (拨备覆盖率 - 行业均值) / 拨备覆盖率 × 100%)
            还原金额 = 当年计提拨备 × 还原比例
        else：
            无需还原

Step C：计算还原 EPS
        还原EPS_Bank = 报表EPS + (还原金额 × 75% ÷ 总股本)
        # 75% 为边际税率
```

### 第二步：PB 估值锚点校验 + 估值修复溢价

核心原则：当 PB 处于历史底部时，即使 EPS 零增长，估值修复本身就能产生显著回报。

```
Step A：计算 PB 历史分位数
        当前PB分位数 = rank(当前PB, PB_历史序列) / len(PB_历史序列) × 100%

Step B：判断是否触发 PB 锚点
        if 当前PB < 0.5 倍：触发条件：PB < 0.5 倍（极低估）
        elif 当前PB分位数 < 15%：触发条件：PB 历史分位数 < 15%
        else：不触发

Step C：计算估值修复溢价
        if 触发条件存在：
            if 当前PB < 0.3 倍：修复溢价系数 = 20%
            elif 当前PB < 0.4 倍：修复溢价系数 = 15%
            elif 当前PB < 0.5 倍 或 PB分位数 < 15%：修复溢价系数 = 10%
            修复溢价 = DDM安全买入价 × 修复溢价系数
            PB调整后安全买入价 = DDM安全买入价 + 修复溢价
        else：
            PB调整后安全买入价 = DDM安全买入价
```

### 第三步：动态 Target Yield（银行类债属性）

核心原则：资本充足率远高于监管要求 + 历史分红极稳 → 类债属性更强 → 可降低目标收益率。

```
Step A：评估资本充足率安全边际
        核心一级资本充足率要求 = 5%（最低监管要求）
        安全边际 = 实际核心一级资本充足率 - 最低要求

Step B：判断是否满足下调条件（需同时满足）：
        ① 核心一级资本充足率 > 9%（安全边际充足）
        ② 近5年分红率标准差 < 5pp（分红极稳定）
        ③ 近3年核心一级资本充足率无显著下滑趋势

        if 满足全部条件：
            if 核心一级资本充足率 > 11%：
                target_yield_bank = 6.0%（类债属性极强）
            else：
                target_yield_bank = 6.5-7.0%（稳健下调）
        else：
            target_yield_bank = Phase 2.4 确定的 target_yield（标准处理）

Step C：Phase 2.7的银行专项下调优先生效，不与Phase 2.4上调重复叠加
```

### 第四步：银行股安全买入价最终取值

核心原则：银行股的安全买入价 = **DDM结果** 与 **股息率锚定价** 的较高者。

```
Step A：计算股息率锚定价
        股息率锚定价 = (EPS × 分红率) / target_yield_bank

Step B：取两者较高值
        if Phase 2.7 触发了 PB 修复溢价：
            安全买入价 = max(DDM安全买入价 + 修复溢价, 股息率锚定价)
        else：
            安全买入价 = max(DDM安全买入价, 股息率锚定价)
```

### Phase 2.7 执行流程总结

```
Step 1：识别银行股 → Phase 2.5 识别"PB修复+股息"路径
Step 2：获取银行专项数据（B①资本充足率、B②拨备覆盖率、B③近5年PB）
Step 3：坏账拨备还原 → 超额拨备 → 还原EPS_Bank / 不超额 → 报表EPS
Step 4：PB锚点校验 → PB < 0.5倍 或 分位数 < 15% → 叠加10-20%溢价 / 否则无溢价
Step 5：动态Target Yield → 资本充足率充足+分红极稳 → 下调至6-7% / 否则维持
```
