# 内生增长分红估值模型（V1.0）—— 模型规格文档

## 1. 核心估值公式

```
安全买入价 = [ EPS_curr × (1 + g_final)^10 × Payout ] / 8%
```

**参数定义：**

| 参数 | 含义 | 说明 |
|------|------|------|
| `EPS_curr` | 最新年报每股收益 | 必须使用最新年报数据（非 TTM）；年报未发布时才降级使用 TTM |
| `Payout` | 分红率 | **优先使用公司明确的分红承诺/规划**（如"分红率不低于75%"）；无承诺时用近3-5年均值 |
| `ROE_5y` | 近5年平均 ROE | 使用加权净资产收益率（年报数据） |
| `10` | 复利计算周期 | 10 年 |
| `8%` | 目标成本收益率 | Yield on Cost = 8% |

---

## 2. g_final（内生增速）的嵌套计算逻辑

**第一步：计算理论增速**

```
g_base = ROE_5y × (1 - Payout_5y)
```

**第二步：根据 ROE 档位强制截断，防止"增长幻觉"**

| ROE 档位（5年平均） | 生意类型 | 增速上限（Cap） | 逻辑参考 |
|-------------------|----------|---------------|---------|
| ROE > 25% | 顶级生意 | **8%** | 护城河深，再投资回报极高 |
| ROE 15%-25% | 强力生意 | **5%** | 行业较饱和，增长受限 |
| ROE < 15% | 平庸/公用事业 | **3%** | 增长缓慢，接近 GDP 或通胀 |

```
g_final = min(g_base, Cap)
```

**第三步：评估 g_final 可信度（Phase 2.5）**

g_final 是模型对"未来 10 年 EPS 复合增速"的核心假设。**理论推导的增速 ≠ 实际能实现的增速**，必须从四个维度检验其可信度：

| 维度 | 核心问题 | 逻辑 |
|------|---------|------|
| 历史增速 | 过去 EPS 实际增速能否支撑 g_final？ | 如果历史 CAGR 远低于 g_final，说明 g_final 是乐观假设 |
| 再投资效率 | 留存利润再投资能否真的产生 ROE 水平的回报？ | 高分红公司（≥70%）不依赖再投资，g_final 天然容易实现 |
| 行业空间 | 行业还有没有增长空间？ | 存量竞争行业的公司难以维持增速 |
| ROE 稳定性 | ROE 能否持续保持在当前水平？ | ROE 波动大 → 未来盈利不确定性高 |

详见 `SKILL.md` Phase 2.5 的完整评分规则。

## 3. 适用范围与筛选条件（一票否决）

满足以下全部条件，方可使用本模型估值：

1. **历史分红稳定**：最近 5 年，每年分红率均 > 30%
2. **现金流健康**：过去 3 年平均经营现金流净额 ≥ 净利润
3. **负债率可控**：资产负债率平稳或下降，严禁"借钱分红"
4. **业务确定性**：商业模式简单，10 年后该产品/服务大概率仍被需要

## 4. 使用要点

- **宁可保守**：永远给预测留出余地
- **分红是"谎言检测器"**：连续高分红能过滤虚假利润
- **动态调整分母 r**：极其确定的资产（如长电）可微调至 6%-7%
- **银行股的特殊回报路径**：招商银行 2023-11 回测验证，银行股低估时实际上涨驱动力可能是估值修复（PB 修复）+ 高股息率，而非 EPS 增长。模型安全价作为"价值锚"有效，但需补充分析 PB 位置和股息率
- **模型对低增长股票的局限**：当 EPS 近乎零增长时，模型假设的增长路径不成立，但安全买入价仍可判断低估/高估区间。低估判断的准确性来源于安全边际，而非增长预测
- **品牌溢价突破模型框架**：贵州茅台 2020-11 回测验证，品牌护城河极深的公司（"类货币"属性）长期享有估值溢价，市场价可持续远超安全买入价。模型安全价更像"价值底线"而非"合理价格"。极端高估（PE 45+）虽然不会立即下跌，但长期确实损害回报（茅台 5 年零收益验证）
- **模型的核心价值是纪律性**：在极端高估时发出"不追涨"信号，在明显低估时提供"价值锚"参考。模型无法预测价格拐点和市场情绪延续时间
- **安全边际负值需分级解读**：贵州茅台 2018-11 回测首次出现"高估误判"——PE 28、安全边际 -103% 被判为"明显高估"，但 7.4 年后总回报 172.7%。教训：安全边际 -50% 至 -100% 不等同于极端高估，需配合 PE 历史分位数综合判断。熊市恐慌中（全市场普跌 >20%）的"高估"信号应格外谨慎
- **顶级公司增速假设可能过保守**：ROE 持续 >25% 且分红率 50%+ 的公司，实际 EPS 增速可达 15%+，模型 8% 增速上限对此类公司偏保守。但这属于"宁可保守"的代价，不建议放松

> 注：本模型仅供参考，不构成投资建议。

## 5. 计算示例

假设某公司：
- EPS_curr = 5.00 元
- ROE_5y = 22%（强力生意）
- Payout_5y = 65%

计算过程：
1. `g_base = 0.22 × (1-0.65) = 0.22 × 0.35 = 7.7%`
2. ROE 在 15%-25%，Cap = 5%，故 `g_final = min(7.7%, 5%) = 5%`
3. `未来10年EPS = 5.00 × (1+5%)^10 = 5.00 × 1.6289 = 8.14元`
4. `未来10年DPS = 8.14 × 0.65 = 5.29元`
5. `安全买入价 = 5.29 / 8% = 66.16元`
